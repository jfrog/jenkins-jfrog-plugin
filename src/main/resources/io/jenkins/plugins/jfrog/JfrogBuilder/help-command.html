Enter the JFrog CLI command you want to execute. The command must start with jf or jfrog.

Examples:

  • jf rt ping - Test connection to Artifactory
  • jf rt u target/*.jar my-repo/ - Upload JAR files to a repository
  • jf rt dl my-repo/artifact.jar - Download an artifact
  • jf rt bp - Publish build-info
  • jfrog docker push my-image:latest - Push a Docker image
  • jfrog mvn clean install - Execute Maven build with JFrog integration

Note: You can use either jf or jfrog as the command prefix.
